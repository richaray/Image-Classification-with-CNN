# 🧠 Image Classification with CNN

An end-to-end image classification pipeline built with Convolutional Neural Networks,
leveraging Transfer Learning for high accuracy with minimal training time.

## ✨ Features

- **Transfer Learning**    → Fine-tuned MobileNetV2 for efficient and accurate classification
- **Data Augmentation**    → Improves generalization by expanding training data variety
- **OpenCV Integration**   → Handles image preprocessing and real-time input handling
- **Evaluation Metrics**   → Accuracy, confusion matrix, and more for model performance analysis

---

## 🛠️ Tech Stack

- Python, PyTorch / TensorFlow
- MobileNetV2 (pretrained)
- OpenCV
- Matplotlib / Seaborn (visualization)

---

## 💡 Use Cases

- Object recognition
- Medical image analysis
- Custom dataset classification
